# 📊 Player Profiles Bot - Quick Guide

## Finding Players
Type `!Player Name` in this thread to find or create a profile.
- **Existing player:** Bot links to their profile thread
- **New player:** Bot creates a profile (limit: 5 requests per day)

Examples: `!Juan Soto` `!Bobby Witt Jr` `!Paul Skenes`

---

## Profile Thread Commands
Use these commands in any player's profile thread:

### **!stats**
Shows current 2026 stats + underlying metrics (Barrel%, xERA, K%, etc.)
*Once per day*

### **!outlook** 
AI-generated rest-of-season projection analyzing performance and what to expect
*Once per day* • *Available April 15+*

### **!splits**
Home/Away, vs LHP/RHP, Day/Night performance splits
*Once per day*

### **!trends**
Recent performance trends vs season averages
*Once per day*

---

## ⏱️ Cooldowns
Each command can be used **once per day per player thread** to keep the bot responsive for everyone.

## 🎯 Tips
- Commands only work in player profile threads (not this search thread)
- All outlooks are AI-generated analyzing 2026 stats so far
- Top 250 players by ADP already have profiles ready
- Admins have no rate limits
